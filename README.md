<br/>

I design interfaces and then build them. Most of my work lives in the handoff
gap — the place where a clean Figma file usually turns into messy code. I try to
close that gap myself: brand and product design on one side, React and motion on
the other.

Lately I've been into **real-time 3D on the web** (Three.js, React Three Fiber)
and **design systems** that survive contact with engineering.

<br/>

**Design** &nbsp;·&nbsp; Figma · Framer · After Effects · Illustrator · Photoshop
**Build** &nbsp;·&nbsp; React · Three.js · Framer Motion · TypeScript · Tailwind

<br/>

### Selected work

**Interactive 3D Portfolio**
A portfolio that runs live 3D scenes in the browser with motion that doesn't
get in the way. Built with React, Three.js and Framer Motion.

**Aether — Mobile Experience**
End-to-end product and brand design for a mobile flagship: research,
flows, UI, and the design system behind it.

**Brand Identity Systems**
Logos, guidelines and scalable design systems for early-stage teams.
